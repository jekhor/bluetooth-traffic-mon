   БОльшая часть содержимого данных библиотек взята из Интернета после формального отбора по нескольким критериям пригодности вообще и для ГОСТ в частности. Практической проверке подверглась лишь малая часть содержимого, по этой причине рекомендуется перепроверять библиотечный элемент перед его первым использованием.
   Нарисование УГО не всегда полностью совпадает с ГОСТ, поэтому, если нормоконтроль лютует, требуется некоторая доводка элементов библиотеки схематика для приведения в соответствие требованиям ГОСТа.
   Библиотека Additional.mod обычно не подключается чтобы не перегружать и без того длинный список, а используется только для пополнения библиотеки Main.mod по необходимости.
   Библиотека packages3d содержит гораздо больше элементов, чем активно используется. Например, 3d-элементы имеющие разные цвета, чаще всего представлены только одним цветом, а остальные можно подключать по необходимости. Также, в этой библиотеке есть очень много компонентов не связанных с посадочными местами, которые не включены в mod-библиотеку. Таким образом, рекомендуется посещать 3d-библиотеку при необходимости найти что-то дополнительное. Лучше всего просматривать эту библиотеку сторонним вьюером, например, SwirlX3D Viewer v.3.2.0 в бесплатной версии.
   Файл kicad.pro позволяет прописать эти библиотеки, чтобы кикад не ругался на их отсутствие.
