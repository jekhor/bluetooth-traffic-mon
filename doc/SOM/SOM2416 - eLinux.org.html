KIT2416-II		 KIT2416-I and KIT2416-II is a NECESSARY basic evaluation system kit for illustrating the functionality, connectivity of SOM2416-I/SOM2416-II modules on their each App Demo Board.
each KIT2416 includes 
  1. SOM2416-I(74pin,4x4cm) or SOM2416-II(120pin,4x4cm) Embedded Module
  2. 4.3" 16:9 TFT LCD, LED backlight digital adjustable, 480 x RGB x 272 pixels, with 4-wire resistance touch screen
  3. Theirs corresponding App Demo Board Rev 1.1, which having USB host and device/Ethernet/3 or 4 UART/IIC/SPI/LCD/touch screen/Stereo Audio out/MIC/Line in/SDMMC/ADC/PWM/GPIO/local bus interfaces
  4. Cables : Ethernet cable, mini USB cable, RS232 Cable
  5. Power Supply AC 110V~220V in DC 5V 1A output
  6. SDK: WINCE 5.0 SDK and documents,App Demo Board's SCH, PCB source code; Linux SDK in under developing and planing Nov 2009 release
